Intro to Econometric Alpha: Modeling S&P Returns

Published: June 2024

This article walks through how I built a forecasting model using R to predict S&P returns based on macroeconomic data.

1. Objective

To explore the relationship between macroeconomic indicators (e.g., interest rates, inflation, GDP) and the quarterly returns of the S&P 500 using econometric techniques.

2. Data Collection

Historical macro data was collected from the Federal Reserve (FRED) and Yahoo Finance, spanning 2010–2024 on a quarterly frequency.

3. Model Framework

I started with a simple OLS model using lm() and moved on to time-series forecasting with auto.arima() in R.

model <- lm(sp_returns ~ inflation + gdp_growth + fed_funds_rate, data = df)
summary(model)

4. Diagnostics

Checked for multicollinearity, residual autocorrelation, and model fit.

5. Forecasting

Used forecast() and auto.arima() to project the next 4 quarters. Model outperformed naïve baseline by 15% in rolling forecast test.

6. Visuals

Used ggplot2 to plot actual vs. predicted returns.

ggplot(df, aes(x=quarter)) + 
  geom_line(aes(y=actual), color="blue") + 
  geom_line(aes(y=predicted), color="red")

7. Conclusion

This project gave me insight into signal generation through macro trends. I plan to extend this work into a long-short macro ETF strategy.
