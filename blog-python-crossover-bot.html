Python for Traders: Moving Average Crossover Bot

Build a simple yet powerful trading bot in Python using SMA and EMA crossover strategies

🎯 Strategy Overview

We use a crossover strategy where the bot buys when the short-term moving average crosses above the long-term average (bullish signal), and sells when it crosses below (bearish signal).

🔍 Data Collection

Using yfinance to download historical price data:

import yfinance as yf
data = yf.download('AAPL', start='2022-01-01', end='2023-01-01')

📈 Bot Logic

Here’s the core logic for detecting crossovers and simulating trades:

data['SMA_20'] = data['Close'].rolling(window=20).mean()
data['SMA_50'] = data['Close'].rolling(window=50).mean()
data['Signal'] = 0
data['Signal'][20:] = np.where(data['SMA_20'][20:] > data['SMA_50'][20:], 1, -1)
