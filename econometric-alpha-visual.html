Forecasting S&P 500 Returns using Linear Regression

This visual illustrates the comparison between actual and predicted S&P 500 returns. The model was built using macroeconomic indicators such as the Unemployment Rate and Consumer Price Index (CPI), employing a linear regression approach to forecast returns over time.

By analyzing these indicators, we aim to identify patterns that can help estimate future movements in the equity markets. The model demonstrates how econometrics can be applied to real-world trading insights.
